# qq-photo-download
qq 空间相册下载脚本

心血来潮，想要整理一把老照片，都放到一处管理。有些老照片是放在qq空间上的，找了网上基本没什么软件，有的也是坑。索性自己写个python+selenium 脚本搞一搞

# tips
1. 用户名和密码 建一个conf.py 文件,格式如下
name = 'xxx'
passwd = 'xxx'
